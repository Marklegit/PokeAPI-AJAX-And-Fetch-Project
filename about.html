PkéAPI

About

Pokémon from generations 1-7. This website allows you to get data of a Pokémon you search. You can find their Pokédex number, type, abilities, base stats, and moves. This website offers a better user experience for someone with prior knowledge of the Pokémon video games, as the PokéAPI is made based on them.

For newcomers! Don't worry if you are new and don't know any Pokémon, you can start by clicking All Pokémon. You can see 807 of them!

If you want to see the code check my GitHub.

How to use search?

Type a Pokémon name and click the search button or click enter.

Make sure you spell the name correctly or you will NOT get any data. Some Pokémon are not available through this API (Application Programming Interface). See below in the Unavailable Pokémon section.

Finding Alternate Forms

To find alternate forms you can use hyphen ( - ) or space. For example Charizard-Mega-X.

To find Nidoran♂Male and Nidoran♀Female add a hyphen/ space and the first letter of the gender. Type Nidoran-M or Nidoran-F. They are special Pokémon who share the same name, but different gender.

Unavailable Pokémon

Pokémon that are not available in the PokéAPI include:

  1. Meltan
  2. Meltmetal
  3. All Pokémon from Generation 8 (Galar)

Missing Artwork or/and Sprites

The PokéAPI does not have official artwork and so I'm using another API to try and get artwork on all Pokémon. Some Mega or alternate forms may be missing their artwork.

Inaccurate information

The Pokédex number for a mega Pokémon is inaccurate. Refer to its original form for its true Pokédex Number.
